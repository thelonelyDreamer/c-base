### 1. 判断某个整数是否是素数

```c
/**
 * 判断一个大于1的正整数是否为素数
 * @return -1 代表输入参数不合理 
 *         0  代表不是素数
 *         1  代表是素数
 */
int fun(const int n){
    if(n<=1) return -1;
    int k,yes=1;
    for(k=2;k<=n/2;k++){
        if(n%k==0) yes=0;
        else yes=1;
    }
    return yes;
}
```
